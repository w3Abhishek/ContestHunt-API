Welcome to the ContestHunt API

Available Endpoints:

  • /contests - List of contests
  • /hackathons - List of hackathons
  • /bounties - List of bounties

This API provides details about coding contests, hackathons, and paid bounties from various platforms.

Use these endpoints to fetch data programmatically for your projects or applications.
